## 项目介绍
基于SSM框架的酒店管理系统，主要功能如下
前台功能:
客房预订、入住办理、登录注册

后台功能：
会员信息管理、客房信息管理、预订信息管理、入住信息管理、公告信息管理、留言板管理

## 项目技术
- 编程语言：Java
- 数据库：MySQL
- 前端技术：JSP、JavaScript、bootstrap、JQuery
- 后端技术：Spring、SpringMVC、MyBatis

## 运行环境
- JDK版本：JDK1.8及以上
- 开发工具：IDEA、Ecplise、Myecplise都可以
- 数据库: MySQL5.7及以上
- Maven：maven3.0及以上
